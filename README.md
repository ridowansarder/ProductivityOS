# Productivity OS

> A personal academic productivity system built for students who want clarity, not complexity.

**[Live Demo](https://productivity-os-five.vercel.app/)**

---

## Key Features

### **Course Management**
- Create and organize courses by semester
- Archive courses when completed (never lose data)
- Unique course validation (no duplicates per semester)

### **Smart Assignment System**
- Assignments linked to courses
- Priority levels (Low, Medium, High)
- Status tracking (Todo, In Progress, Done)

### **Notes System**
- Course-linked notes
- Rich text editing
- Archive instead of delete

### **Archive-First Safety**
- Soft delete by default (archive)
- Restore archived items anytime
- Hard delete only with confirmation

### **User Experience**
- Mobile-responsive sidebar
- Dark/Light mode support
- Empty states
- Confirmation dialogs for destructive actions
- Real-time data updates

## Tech Stack

### **Frontend**
- **Next.js 16** (App Router) - React framework with SSR
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Shadcn UI** - Accessible component library
- **Lucide Icons** - Modern icon set

### **Backend**
- **Next.js Server Actions**
- **Prisma ORM** - Type-safe database client
- **PostgreSQL** - Relational database
- **Clerk** - Authentication & user management

### **Deployment**
- **Vercel** - Frontend hosting
- **Neon/Supabase** - PostgreSQL hosting
- **GitHub Actions** - CI/CD (optional)
